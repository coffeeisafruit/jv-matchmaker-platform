# JV Matchmaker Platform

A comprehensive JV partnership platform that combines AI-powered matching, data enrichment, and launch content generation.

## Vision

Build a full-stack platform that helps entrepreneurs find, pitch, close, and execute JV partnerships with a 54-step execution playbook and AI-powered launch content generation.

## Tech Stack

- **Backend**: Django
- **Frontend**: HTMX + Alpine.js + Tailwind CSS (HAT Stack)
- **Database**: Supabase (1000+ existing profiles)
- **AI**: OpenAI GPT-4o-mini / Anthropic Claude
